    As its name suggests, a Codec[A, B], defines how to decode an A value to a B value, and, conversely, how to encode a B value to an A value.

    The decoding and encoding processes are defined by the decode and encode methods.

    To get an instance of Codec, start with the provided concrete subclasses (see the companion object) and combine them with the andThen, orElse and opt methods.
